## 1. Project Title

**HR Attrition Analysis | Excel, SQL, Python & Power BI**

---

## 2. Project Overview

This project analyzes employee attrition to identify workforce trends, understand the factors influencing employee turnover, and support HR decision-making through interactive dashboards and data-driven insights. :contentReference[oaicite:0]{index=0}

---

## 3. Business Problem

Employee attrition increases recruitment costs, reduces productivity, and affects workforce stability. The objective is to identify key attrition drivers and support proactive employee retention strategies. :contentReference[oaicite:1]{index=1}

---

## 4. Business Objectives

- Identify key attrition drivers.
- Analyze employee turnover trends.
- Evaluate salary and experience patterns.
- Assess the impact of overtime and work-life balance.
- Support HR with actionable insights. :contentReference[oaicite:2]{index=2}

---

## 5. Project Workflow

Business Understanding → Data Cleaning → SQL Analysis → Python EDA → Power BI Dashboard → Business Insights → Recommendations :contentReference[oaicite:3]{index=3}
